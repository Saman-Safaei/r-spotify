# Getting Started with R-Spotify

```shell
$ git clone https://github.com/Saman-Safaei/r-spotify.git
$ cd r-spotify
```
#### with NPM
```shell
$ npm install
$ npx msw init ./public
$ npm start
```
#### with Yarn
```shell
$ yarn install
$ yarn msw init ./public
$ yarn start
```

With this commands the R-Spotify will be started.
